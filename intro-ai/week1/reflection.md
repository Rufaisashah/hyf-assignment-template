Q3:Your 3 learnings from Part A:

- The "Fixed" Requirement: I learned that for a navigation bar to stay visible during a scroll, position: fixed is the anchor that holds it in place.

- Width Reset: I learned that when you switch an element to fixed, it loses its default width, so you have to explicitly set width: 100% to make it span the page.

- Layer Priority: I learned that z-index: 1000 is my way of telling the browser "This is the most important layer," ensuring it never gets buried behind the geometric background pattern.

- State Control: I learned that CSS can control the 'state' of an animation (running vs. paused) based on user behavior like hovering.

- Performance Efficiency: I learned that CSS animations are generally better for performance than JavaScript animations because the browser can optimize them more effectively.

- Visual Feedback: I learned that a 'floating' icon acts as a silent invitation to the user, signaling that the element is an active link they can interact with.


Q3: Your 3 ethical issues and mitigations from Part B:


1. Issue: AI "Hallucinations." Sometimes AI gives code that works on one screen but breaks on another.

Mitigation: I will use the "Inspect Element" tool in my browser to toggle between Mobile and Desktop views to ensure the code works for everyone.

2. Copy-Paste Errors. Pasting CSS over existing CSS can leave "ghost" properties (like two different background-colors) that cause bugs.

Mitigation: I will clear my CSS file completely before pasting a structural update to ensure there are no conflicting rules from previous versions

3. Mobile Responsiveness. A layout that looks good on 100vh might be completely broken on a small phone screen where the keyboard pops up.

Mitigation: I will use Media Queries to change 100vh to auto height on mobile devices, ensuring the user can always scroll to see the content.

4. Issue: Code Ownership Confusion. AI can generate complex variables that mimic a specific copyrighted brand's theme.

Mitigation: I will use the AI to understand the system of variables, but I will choose my own custom hex codes to ensure my design is original.
